
### **1. Frontend (フロントエンド)**

主に管理者画面の拡張と、スーパー管理者向けの新設ページが中心です。

- **スーパー管理者機能の追加 (
    
    ```
    app/super-admin/
    ```
    
    )**
    - ```
        app/super-admin/page.tsx
        ```
        
        : スーパー管理者用のトップページを作成。
    - ```
        app/super-admin/operations/page.tsx
        ```
        
        : オペレーション管理ページを追加。
    - ```
        app/super-admin/sales/page.tsx
        ```
        
        : 営業（セールス）管理ページを追加。
    - ```
        app/super-admin/layout.tsx
        ```
        
        : スーパー管理者エリアのレイアウト定義。
- **既存管理者画面の改修**
    - ```
        app/admin/dashboard/page.tsx
        ```
        
        : ダッシュボードの機能更新。
    - ```
        app/admin/settings/page.tsx
        ```
        
        : 設定画面の更新。
    - ```
        components/admin/DashboardClient.tsx
        ```
        
        : ダッシュボードのクライアントコンポーネント更新。
    - ```
        components/store/StorePortalUI.tsx
        ```
        
        : 店舗ポータルUIの調整。
- **型定義・API**
    - ```
        lib/api.ts
        ```
        
        , 
        
        ```
        types/index.ts
        ```
        
        : 新機能に伴うAPIクライアントと型定義の更新。

### **2. Backend (バックエンド)**

データベース設定、スキーマ変更、および新機能（決済・インポート）の実装が含まれます。

- **Store（店舗）スキーマの拡張**
    - src/api/store/content-types/store/schema.json: 店舗情報に 
        
        ```
        status
        ```
        
        （LEAD, ACTIVEなど）、
        
        ```
        salesLogs
        ```
        
        （営業ログ）、
        
        ```
        adminToken
        ```
        
        、
        
        ```
        branding
        ```
        
         などのフィールドを追加。
- **決済機能の追加 (
    
    ```
    src/api/payment/
    ```
    
    )**
    - ```
        controllers/payment.js
        ```
        
        , 
        
        ```
        controllers/webhook.js
        ```
        
        , 
        
        ```
        routes/payment.js
        ```
        
        : 決済処理およびWebhook用コントローラーとルートの新規作成。
- **設定関連**
    - config/database.ts: データベース接続設定の更新（パス解決ロジックなど）。
    - ```
        config/middlewares.ts
        ```
        
        , 
        
        ```
        config/server.ts
        ```
        
        : サーバーおよびミドルウェア設定の調整。
- **スクリプト・ツール**
    - ```
        scripts/import-stores.js/ts
        ```
        
        : 店舗データの一括インポート用スクリプト。
    - ```
        scripts/check-admin-email.js
        ```
        
        , 
        
        ```
        verify-data.js
        ```
        
        : データ検証用スクリプト。
    - ```
        stores.txt
        ```
        
        , 
        
        ```
        import-errors.csv
        ```
        
        : インポート処理に関連するデータファイル。

ご希望であれば、特定の機能（例：スーパー管理者画面の実装詳細や、決済機能の仕組み）について詳しく解説します。
