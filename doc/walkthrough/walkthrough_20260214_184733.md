# ウォークスルー: バックエンド・コンフリクト解消とビルド正常化

このドキュメントでは、バックエンド codebase における Git マージコンフリクトおよび TypeScript ビルドエラーを完全に解消し、安定したビルド環境を復元した手順を記録します。

## 実施内容の概要

1. **コンフリクトマーカーの一掃**:
   - サービス、コントローラー、ルート定義、メールテンプレートから残存していたすべての Git コンフリクトマーカー（`<<<<<<<`, `=======`, `>>>>>>>`）を除去しました。
2. **Strapi v5 スキーマの整合性修復**:
   - `contentTypes.d.ts` および各 API の `schema.json` における二重定義や破損を修復しました。
3. **TypeScript ビルドエラーの修正**:
   - シードデータ（`src/index.ts`）や自動生成定義における属性名の不整合（`baseCapacity` vs `capacity`）を解消しました。
4. **ビルドプロセスの安定化**:
   - 重複定義やパス不整合のあった保守用スクリプトを `tsconfig.json` の `exclude` に追加し、コアシステムのビルドを確実に成功させました。
