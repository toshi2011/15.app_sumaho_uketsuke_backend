# Excel風高密度テーブルリファクタリング実装計画

## 目的
開発者ダッシュボードをカード型レイアウトからExcel風の高密度テーブルに変更し、100店舗以上の状況を一目で把握できるようにする。

## 変更内容

### バックエンド (Strapi)
#### [MODIFY] [store schema](file:///e:/15.app_sumaho_uketsuke/backend/src/api/store/content-types/store/schema.json)
- `isVisuallyChecked` (Boolean) フィールドを追加（手動での目視確認フラグ）
- `visualCheckedAt` (DateTime) フィールドを追加（目視確認日時の記録）

### フロントエンド (Next.js)
#### [MODIFY] [types/index.ts](file:///e:/15.app_sumaho_uketsuke/frontend/types/index.ts)
- `Store` インターフェースに `isVisuallyChecked` と `visualCheckedAt` を追加

#### [MODIFY] [Developer Dashboard](file:///e:/15.app_sumaho_uketsuke/frontend/app/developer/page.tsx)
- **完全リファクタリング**: カード型レイアウトを廃止し、shadcn/ui Tableを使用した高密度テーブルに変更
- **列構成**:
  1. 店舗名（名前 + リンク）
  2. ステータス（Badge表示）
  3. 目視OK（トグルチェックボックス + 確認日時記録）
  4. ロゴ（ヒートマップ）
  5. カバー（ヒートマップ）
  6. メニュー（ヒートマップ）
  7. 営業時間（ヒートマップ）
  8. 住所/Map（ヒートマップ）
  9. 店主属性（ポップオーバー編集 + AI相談ボタン統合）
  10. 利用媒体（アイコン表示 + ポップオーバー編集）
  11. アクション（クイックアクションアイコン + ドロップダウン）
- **ヒートマップロジック**:
  - データあり: 緑背景 (`bg-green-50`) + CheckCircleアイコン
  - データなし: 赤背景 (`bg-red-50`) + AlertCircleアイコン
- **インタラクティブ機能**:
  - 目視確認: チェックON時に `visualCheckedAt` を自動記録
  - 店主属性: クリックでポップオーバーを開き、インライン編集 + 最下部に「🤖 AI用プロンプトをコピー」ボタン
  - 利用媒体: クリックでポップオーバーを開き、チェックボックスで編集
  - アクション列:
    - [📩] SNS/郵送DM記録（アイコンボタン）
    - [📞] 電話記録（アイコンボタン）
    - [...] その他（訪問、お断り、詳細編集）- ドロップダウンメニュー
- **パフォーマンス**: 100行以上でも快適に動作するよう最適化

## 検証計画
### 手動検証
1. `/developer` にアクセス
2. テーブルが全列表示されることを確認
3. ヒートマップの色分け（緑/赤背景）を確認
4. 目視確認トグルの動作と日時記録を確認
5. 店主属性ポップオーバー編集とAI相談ボタンを確認
6. 利用媒体ポップオーバー編集を確認
7. アクション列のクイックアクションアイコンとドロップダウンを確認
8. 複数店舗でのパフォーマンスを確認
