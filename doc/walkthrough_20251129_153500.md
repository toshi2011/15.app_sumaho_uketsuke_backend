# ウォークスルー: 店舗一括インポートスクリプト

CSVファイルから店舗データを読み込み、Strapiに一括登録するスクリプトを実装しました。

## 実装内容

### 1. スクリプト (`backend/scripts/import-stores.js`)
- **機能**:
  - `backend/scripts/stores.txt` (CSV形式) を読み込み
  - トランザクション管理下で店舗とユーザーを作成
  - 重複チェック (電話番号、メールアドレス)
  - `adminToken` (UUID) の自動生成
  - エラー発生時は `import-errors.csv` に出力

### 2. データモデル変更
- **Store**: `adminToken` (Private) フィールドを追加

## 使用方法

1. **準備**:
   - `backend/scripts/stores.txt` にインポートしたいデータをCSV形式で配置します。
   - ヘッダー: `name,email,phoneNumber,address,description,logo_path,cover_path,gender,age_range,platforms`

2. **実行**:
   - Strapiサーバーが起動している場合は停止します。
   - 以下のコマンドを実行します:
     ```bash
     node scripts/import-stores.js
     ```

3. **結果確認**:
   - コンソールに成功・失敗件数が表示されます。
   - 失敗したデータは `backend/import-errors.csv` に保存されます。
