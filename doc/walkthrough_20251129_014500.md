# ウォークスルー: Sales CRM機能強化（利用媒体記録 & アクション細分化）

開発者ダッシュボードに、店舗の「利用媒体（デジタルプレゼンス）」を記録する機能と、営業アクションの種類を細分化する機能を追加しました。これにより、AIへの相談がより具体的で実用的になります。

## 実装内容

### 1. バックエンド (Strapi)
- **`Store` スキーマ**: `digitalPresence` (JSON) フィールドを追加。
  - 利用媒体: 独自HP、Instagram、TikTok、食べログ、ホットペッパー、ぐるなび、Retty、Googleマップ、電話のみ

### 2. フロントエンド (Developer Dashboard)

#### A. 利用媒体チェックボックス
各店舗カードに「利用媒体」セクションを追加し、主要な9つのプラットフォームをチェックボックスで管理できるようにしました。

#### B. アクションボタンの細分化
従来の曖昧な「DM送信」ボタンを廃止し、以下の4種類に細分化しました:
- **[📱 SNS DM]**: Instagram/TikTok等のダイレクトメッセージ
- **[📮 郵送DM]**: 紙のDM・チラシの送付
- **[📞 電話(不在)]**: 架電したが不在
- **[📞 電話(会話済)]**: 架電して会話成功
- **[👣 訪問]**: 直接店舗へ訪問

#### C. 強化されたAIプロンプト生成
「AIに相談」ボタンで生成されるプロンプトに、以下の情報を追加しました:
- **Web活用状況**: チェックした利用媒体のリスト
- **戦略的指示**: 媒体に応じた訴求ポイント（コスト削減、機会損失、業務効率化など）
- **アクション判断**: 郵送DMかSNS DMか、どちらが適切かの判断依頼

#### D. 媒体アイコン表示
店舗名の下に、利用中の媒体のアイコンを小さく表示し、一目で「どの経済圏にいる店か」が分かるようにしました。

## 検証結果

ブラウザテストにより、以下の動作を確認しました:

3. 「SNS DM」ボタンでのログ保存
4. 「AIに相談」ボタンでの強化されたプロンプトのコピー

## AIプロンプトの例（強化版）

```text
あなたはベテランのセールスコンサルタントです。
...

【ターゲットのWeb活用状況】
現在、以下の媒体を利用しています：
Instagram, 食べログ

【AIへの指示】
上記の活用状況を踏まえて戦略を立ててください。
- 「ホットペッパー」等の有料媒体を使っている場合は、コスト削減を訴求するトーク。
- 「SNS」はやっているが予約機能がない場合は、機会損失を訴求するトーク。
- 「電話のみ」の場合は、業務効率化を訴求するトーク。
- 「郵送DM」を送るべきか、「SNS DM」で攻めるべきかの判断もお願いします。
...
```

## 期待される効果

この機能により、AIへの相談の解像度が劇的に向上します。例えば:

**入力例**: 「60代男性」「利用媒体：電話のみ、Googleマップ」

**AIの回答例**:
> 「この店主は『食べログ』などの営業電話にうんざりしている可能性があります。**『ポータルサイトの勧誘ではありません。Googleマップ経由の予約を、電話に出ずに自動で取る道具です』**という切り口で、**郵送DM**（紙）を送りましょう。IT用語は使わず『留守番電話の代わり』と説明するのが鍵です。」

このように、具体的で実践的なアドバイスが得られるため、営業の心理的ハードルが大幅に下がります。
