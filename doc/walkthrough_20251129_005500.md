# ウォークスルー: 店舗データ一括インポートスクリプト

店舗データをCSVからStrapiに一括インポートするスクリプトを実装しました。

## 実装内容

### 1. スクリプト
- **パス**: `backend/scripts/bulk-import-stores.ts`
- **機能**:
  - CSVファイルの読み込みとパース
  - ユーザー作成（店主アカウント）
  - 画像アップロード（あれば）
  - 店舗データ作成
  - **トランザクション管理**: 1行ごとの処理をトランザクションで囲み、失敗時はロールバック
  - **リジューム機能**: 既存の店舗（名前または電話番号で判定）はスキップ
  - **エラーログ**: 失敗した行を `import_errors_[timestamp].csv` に出力

### 2. 使用方法
バックエンドディレクトリで以下のコマンドを実行します。

```bash
npm run seed:stores -- --file=./path/to/stores.csv
```

## 技術的詳細
- **依存パッケージ**: `csv-parse`, `mime-types`, `ts-node`
- **Strapi初期化**: `createStrapi({ distDir: './dist' })` を使用（実行前に `npm run build` が必要）
