# Ticket: Implement Address Auto-fill via Postal Code

## 概要
郵便番号を入力してボタンを押すだけで、住所を自動入力する機能を実装しました。

## 実装機能
1. **郵便番号入力**: 設定画面（基本情報タブ）に郵便番号入力欄を追加。
2. **住所検索ボタン**: クリック時に住所検索APIを呼び出し。
3. **自動入力**: 取得した住所（都道府県+市区町村+町域）を住所欄にセット。

## 技術的詳細
- **バックエンド**: `Store` スキーマに `postalCode` フィールドを追加。
- **プロキシAPI**: CORS問題を回避するため、Next.js内に `/api/address` エンドポイントを作成し、外部API（zipcloud）へのリクエストを中継。
- **エラーハンドリング**: 該当なしやAPIエラー時のメッセージ表示。
