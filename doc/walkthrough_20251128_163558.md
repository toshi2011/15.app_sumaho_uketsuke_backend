# ウォークスルー: 店舗設定画面の拡張

店舗設定画面を拡張し、基本情報、営業時間、予約設定を一元管理できるようにしました。

## 実装内容

### バックエンド

#### [schema.json](file:///e:/15.app_sumaho_uketsuke/backend/src/api/store/content-types/store/schema.json)
`Store`スキーマに以下のフィールドを追加：
- `address`: 店舗の住所
- `businessHours`: 営業時間情報（JSON型）
  - ランチ営業時間（開始・終了・有効/無効）
  - ディナー営業時間（開始・終了・有効/無効）
  - 定休日（曜日の配列）
  - 特定休業日（日付の配列）

### フロントエンド

#### [index.ts](file:///e:/15.app_sumaho_uketsuke/frontend/types/index.ts)
`Store`インターフェースに`address`と`businessHours`フィールドを追加しました。

#### [api.ts](file:///e:/15.app_sumaho_uketsuke/frontend/lib/api.ts)
`updateStore`関数を新規追加し、店舗情報の更新をサポートしました。

#### [page.tsx](file:///e:/15.app_sumaho_uketsuke/frontend/app/admin/settings/page.tsx)
設定画面を完全にリニューアルし、3つのセクションに分割：

1. **基本情報セクション**
   - 店舗名
   - 紹介文
   - 住所
   - 電話番号

2. **営業時間セクション**
   - ランチ営業時間（トグルで有効/無効切り替え）
   - ディナー営業時間（トグルで有効/無効切り替え）
   - 定休日（曜日ボタンで選択）
   - 特定休業日（カンマ区切りで日付入力）

3. **予約設定セクション**（既存機能を維持）
   - 総席数
   - 30分あたりの最大組数
   - 平均滞在時間

## 使用方法

1. 管理画面の「設定」タブから店舗設定ページにアクセス
2. 各セクションで必要な情報を入力
3. 「設定を保存」ボタンをクリックして保存

これにより、店舗の基本情報と営業時間を簡単に管理できるようになりました。
