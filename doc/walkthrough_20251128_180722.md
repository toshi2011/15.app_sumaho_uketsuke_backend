# ウォークスルー: タブ式店舗設定画面（完全版）

店舗設定画面を大幅にリニューアルし、タブ切り替え式のUIで多機能かつ使いやすい画面を実現しました。

## 実装内容

### バックエンド
`Store`スキーマに以下のフィールドを追加しました：
- **`socialLinks` (JSON)**: 公式HP、Instagram、X、LINEのURLを管理
- **`branding` (JSON)**: テーマカラー、カバー画像表示設定を管理
- **`logoImage` (Media)**: 店舗ロゴ画像
- **`coverImage` (Media)**: 店舗カバー画像

### フロントエンド
設定画面（`/admin/settings`）を以下の5つのタブに分割しました：

1. **基本情報**
   - 店舗名、紹介文、電話番号、住所
   - **Google Mapプレビュー**: 入力した住所の地図をリアルタイムで表示

2. **営業時間**
   - ランチ/ディナーの営業時間設定
   - 定休日、特定休業日の設定

3. **SNS・リンク**（新規）
   - 公式HP、Instagram、X (Twitter)、LINE公式アカウントのURL設定

4. **デザイン**（新規）
   - **テーマカラー**: カラーピッカーでブランドカラーを選択
   - **画像アップロード**: ロゴとカバー画像のアップロード（プレビュー付き）
   - **リアルタイムプレビュー**: 設定内容が実際の店舗カードでどう見えるかを確認可能

5. **在庫設定**
   - 総席数、組数、滞在時間の設定
