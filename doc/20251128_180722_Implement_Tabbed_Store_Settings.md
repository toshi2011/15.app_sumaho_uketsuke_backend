# Ticket: Implement Tabbed Store Settings

## 概要
店舗設定画面（`/admin/settings`）を全面的にリファクタリングし、タブ切り替え式のUIを実装しました。

## 実装機能
1. **タブUI**: 「基本情報」「営業時間」「SNS・リンク」「デザイン」「在庫設定」の5つのタブを実装。
2. **基本情報**: 店舗名、紹介文、電話番号、住所（Google Mapプレビュー付き）の編集。
3. **営業時間**: ランチ/ディナーの有効化・時間設定、定休日、特定休業日の設定。
4. **SNS・リンク**: 公式HP、Instagram、X (Twitter)、LINEのURL設定。
5. **デザイン**: テーマカラーの選択、ロゴ・カバー画像のアップロードとプレビュー。
6. **在庫設定**: 総席数、最大組数、滞在時間の設定。

## 技術的詳細
- **バックエンド**: `Store` スキーマに `socialLinks`, `branding`, `logoImage`, `coverImage` フィールドを追加。
- **フロントエンド**: `useState` によるフォーム状態管理、画像アップロード処理の実装。
